# CartPro ASP.NET Web Application

## Overview
CartPro is an e-commerce web application for fashion accessories including watches, shoes, perfumes, and belts. This project has been converted from HTML/CSS/JS to ASP.NET Web Forms.

## Project Structure
- **Default.aspx** - Home page
- **Shop.aspx** - Product listing page
- **Cart.aspx** - Shopping cart page
- **Checkout.aspx** - Checkout page
- **About.aspx** - About us page
- **Contact.aspx** - Contact page
- **Services.aspx** - Services page
- **Login.aspx** - User login page
- **Registration.aspx** - User registration page
- **ViewProfile.aspx** - User profile page
- **Logout.aspx** - Logout functionality
- **ThankYou.aspx** - Order confirmation page
- **Site.Master** - Master page with header and footer

## Requirements
- Visual Studio 2019 or later
- .NET Framework 4.8
- IIS Express (included with Visual Studio)

## How to Run
1. Open `CartProWebApp.sln` in Visual Studio
2. Press F5 or click "Start" to run the project
3. The application will open in your default browser at `http://localhost:44300/`

## Features
- Responsive design using Bootstrap 5
- Master page layout for consistent header and footer
- Product catalog with multiple categories
- Shopping cart functionality
- User authentication (Login/Registration)
- Contact form
- Testimonial slider
- Profile management

## Technologies Used
- ASP.NET Web Forms (.NET Framework 4.8)
- Bootstrap 5
- Font Awesome 6
- Tiny Slider
- Custom CSS

## Default Page
The default starting page is `Default.aspx` (Home page)

## Notes
- This is a Web Forms project, not MVC
- All static assets (CSS, JS, images) are included in the project
- The project uses IIS Express for local development
- Session management is implemented for user authentication
